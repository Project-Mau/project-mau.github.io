The Mau Markup Language

A simple and powerful template-based markup language

Explore now Star on Github

Mau is a lightweight markup language heavily inspired by Markdown and AsciiDoc that makes it a breeze to write blog posts or books. If you already know Markdown or AsciiDoc you already know which type of software Mau is, and you will quickly learn its syntax.

Main features

Mau includes a lot on components that make it a breeze to create and customise content.

Getting started

With a basic syntax that comes from Markdown and AsciiDoc getting started is a matter of minutes.

HTML and PDF

Two visitors are available out of the box to provide HTML and TeX output.

Templates

Use Jinja to customise the output of each component. Templates have a huge amount of customisation options.

Flexible API

It's easy to create new visitor should you want to use different template engines.

Mau 4.x is production-ready and can be used out of the box to create beautiful HTML pages and PDF documents!
